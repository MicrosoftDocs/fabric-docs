# How-To: Enable item lifecycle notifications

Configure your workload to receive notifications for all item lifecycle events from any source within the Fabric platform.

> [!NOTE]
> Item lifecycle notifications are currently under development and will be available soon in an upcoming release.

## Overview

Item lifecycle notifications provide real-time notifications for all CRUD operations on your items from:

- **UX operations**: User actions through Fabric interface
- **Public Fabric APIs**: Operations via REST APIs
- **Fabric processes**: Internal system operations

## Notification events

- **Create**: New item creation
- **Read**: Item access and viewing
- **Update**: Item modifications and changes
- **Delete**: Item deletion operations

## Configuration

Configuration details will be provided when the feature becomes available.

## Use cases

- **Audit and compliance**: Track all item operations
- **Integration workflows**: Trigger external processes
- **Monitoring**: Usage tracking and analytics
