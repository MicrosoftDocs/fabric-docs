# OneLake security integration reference

The integration reference includes the API specifications for integrating with OneLake security to read data and enforce security. Today OneLake supports APIs for using authorized engine access to data.

## Authorized engine APIs

| API | Description | Status |
| --- | --- | --- |
| [Get authorized access for a principal](./authorized-access-for-principal-reference.md) | Fetches the authorized access that the specified principal has on an item. | Preview |
