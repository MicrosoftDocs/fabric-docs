> [!NOTE]
> Billing for extended capabilities is temporarily paused but CDF and views continue to function as expected for your mirrored databases. UX support for enabling CDF or views is temporarily not available. In the meantime, you can enable or disable [extended capabilities via APIs](../mirrored-database-rest-api.md#json-definition-example-of-replicating-entire-database).
