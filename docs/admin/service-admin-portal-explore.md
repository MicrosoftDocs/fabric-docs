# Tenant settings for Explore

These settings are configured in the tenant settings section of the [Admin portal](./about-tenant-settings.md#how-to-get-to-the-tenant-settings). For information about how to get to and use tenant settings, see [About tenant settings](./about-tenant-settings.md).

## Users with view permission can launch Explore

Explore is a lightweight visual data exploration experience that lets users perform ad hoc analysis on semantic models and on items connected to those semantic models.

Turn on this setting to allow users with view permission on a semantic model to launch Explore from that model and from items connected to it.

For more information, see [Explore data in the Power BI service](/power-bi/consumer/explore-data-service#permissions-and-requirements).
