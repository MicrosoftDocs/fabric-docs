1. Follow one of these steps based on whether you're using Eventstream or Real-Time hub:
    - **Eventstream:** In the **Source details** pane to the right, select the **Pencil** button to change the **Source name**. This step is optional.
    - **Real-Time hub:** In the **Stream details** section, select the **Fabric workspace** where you want to create the eventstream and select the **Pencil** button to enter a **name** for the eventstream.
