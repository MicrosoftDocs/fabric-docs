# Schema Registry region availability (preview)

This article lists the Azure regions where the [Schema Registry](schema-registry-overview.md) feature in Microsoft Fabric Real-Time Intelligence is available. Schema Registry and Event Schemasets are currently in preview.

Your tenant [home region](../../admin/find-fabric-home-region.md) determines whether you can use Schema Registry without extra configuration. If your home region isn't listed here, create a Fabric capacity in one of the supported regions to access the feature. For more information, see [Multi-Geo support for Fabric](../../admin/service-admin-premium-multi-geo.md).
