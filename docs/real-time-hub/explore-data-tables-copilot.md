# Explore KQL tables with Copilot in Real-Time hub

If the data stream is[!INCLUDE [copilot-explore-data](../real-time-intelligence/includes/copilot-explore-data.md)] based on a Kusto Query Language (KQL) table, you can preview, explore, and visualize data in the KQL table, using Copilot:

1. In the list of **Recent streaming data**, filter by the **tables** data type.

1. Scroll to or search for a KQL table in the list and select the eye button to see more options.

1. Select **Explore data** to open the KQL table with Copilot, where you can interactively explore and visualize the data in the KQL table.
