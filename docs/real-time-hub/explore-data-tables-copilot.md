# Explore KQL tables with Copilot in Real-Time hub

If the data stream is based on a Kusto Query Language (KQL) table, you can preview, explore, and visualize data in the KQL table, using Copilot.

## Access Copilot to explore KQL table data

There are three ways to access the **Explore data** option for KQL tables in the Real-time Hub.

**Explore data-in-motion tile**

1. Select the **Explore data-in-motion** tile.
1. Filter by the **tables** data type.
1. Select a KQL table, and then select **Choose**.

**List of Recent streaming data**

1. In the list of **Recent streaming data**, filter by the **tables** data type.
1. From a KQL table stream's context menu (three dots), select **Explore data**.

**KQL table details**

1. Open the KQL table details.
1. Select **Explore data** from the upper tool bar.
