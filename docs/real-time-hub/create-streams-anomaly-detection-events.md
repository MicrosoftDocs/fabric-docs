# Get Fabric anomaly detection events in Real-Time hub

This article describes how to get Fabric anomaly detection events as an eventstream in Fabric Real-Time hub.

Fabric anomaly detection events are events generated by the Fabric Anomaly Detection system that identify anomalies in your Eventhouse data. These events can provide insights into unusual patterns or behaviors in your data, allowing you to take proactive measures to address potential issues.


[!INCLUDE [anomaly-detection-events-connector-prerequisites](../real-time-intelligence/event-streams/includes/connectors/anomaly-detection-events-connector-prerequisites.md)]

## Create streams for anomaly detection events

You can create streams for Fabric anomaly detection events in Real-Time hub using one of the ways:

- [Using the **Data sources** page](#data-sources-page)
- [Using the **Fabric events** page](#fabric-events-page)

## Data sources page

[!INCLUDE [launch-get-events-experience](./includes/launch-get-events-experience.md)]

5. On the **Data sources** page, select **Fabric events** category at the top, and then select **Connect** on the **Anomaly detection events** tile.

    :::image type="content" source="./media/create-streams-anomaly-detection-events/add-data-anomaly-detection-events.png" alt-text="Screenshot that shows the selection of Anomaly detection events as the source type in the Data sources page." lightbox="./media/create-streams-anomaly-detection-events/add-data-anomaly-detection-events.png":::

    Now, use instructions from the [Configure and create an eventstream](#configure-and-create-an-eventstream) section.

## Fabric events page

1. In Real-Time hub, select **Fabric events** on the left navigation menu.
1. Move the mouse over **Anomaly detection events**, and select the **+** (plus) link, or select **... (ellipsis)** and then select **Create Eventstream**.

    :::image type="content" source="./media/create-streams-anomaly-detection-events/fabric-events-page.png" alt-text="Screenshot that shows the selection of Anomaly detection events as the source type in the Fabric events page." lightbox="./media/create-streams-anomaly-detection-events/fabric-events-page.png":::

    Now, use instructions from the [Configure and create an eventstream](#configure-and-create-an-eventstream) section, but skip the first step of using the **Add source** page.
