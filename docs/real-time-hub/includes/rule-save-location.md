## Save location section

1. In the **Save location** section, for **Workspace**, select the workspace where you want the Fabric activator item to be created or that already exists. 

1. For **Item**, select an existing activator item or creat a new one. To create a new activator item, select **Create a new item**, and then enter a **name** for the activator item.
