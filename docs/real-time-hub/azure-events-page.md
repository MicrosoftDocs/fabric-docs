# Explore the Azure events page in Fabric Real-Time hub

Azure events are system events that Azure services generate, such as changes to blobs in Azure Blob Storage. Like Fabric events, they're produced by the platform rather than defined by you, which lets you bring signals from your Azure resources into Real-Time hub alongside your Fabric data.

On this page, you discover the Azure events that you can access and subscribe to by creating an event stream. After you subscribe to an event, you can route it to downstream destinations or configure alerts that send notifications through email, Teams, and other supported channels when an event occurs.
