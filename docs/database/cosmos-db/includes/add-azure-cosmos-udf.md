1. In the top menu bar, select **Library Management**, then choose **+ Add from PyPI**. Search for `azure-cosmos`, select it from the results, and pick the latest version.

   :::image type="content" source="media/how-to-user-data-functions/add-cosmos-library.png" alt-text="Screenshot showing how to add azure-cosmos library." lightbox="media/how-to-user-data-functions/add-cosmos-library.png":::

1. Close the library management pane once the library is added, and on the top menu bar, select **Publish**.

   :::image type="content" source="media/how-to-user-data-functions/publish-user-data-function.png" lightbox="media/how-to-user-data-functions/publish-user-data-function.png" alt-text="Screenshot of the Publish button on the top menu bar.":::
