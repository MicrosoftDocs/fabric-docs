|April 2025|**CI/CD and REST APIs for Fabric Eventstream**|[Eventstream CI/CD and REST APIs](https://blog.fabric.microsoft.com/blog/announcing-the-general-availability-of-ci-cd-and-rest-apis-for-fabric-eventstream?ft=All) are now generally available.
