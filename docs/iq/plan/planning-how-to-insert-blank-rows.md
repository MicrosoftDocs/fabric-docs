# Insert blank rows

In reports and financial statements, you may need to add blank rows to improve readability and formatting. Plan allows you to insert blank rows in table or matrix reports to create visual spacing for better readability and presentation.

In this article, you learn how to insert and remove blank rows.

## Insert a blank row

1. Select the row above which you want to insert a blank row.
1. Hover over the row and select the row gripper.
1. Select **Insert** > **Add Empty Row**.

    :::image type="content" source="media/planning-how-to-insert-blank-rows/insert-blank-row.png" alt-text="Screenshot of steps to insert a blank row." :::

A blank row will be inserted above the selected row. Blank rows can be inserted at any level of the hierarchy.

## Delete a blank row

1. Hover over the blank row and select the row gripper.
1. Select **Remove Empty Row**.

    :::image type="content" source="media/planning-how-to-insert-blank-rows/delete-blank-row.png" alt-text="Screenshot of steps to remove a blank row." :::
