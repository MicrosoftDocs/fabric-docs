# Insert attachment columns in PowerTable

An attachment column is a type of [visual column](how-to-insert-visual-columns.md) that you use to attach and manage files directly within table records. The record stores the attached files, so you can easily associate supporting documents, images, and other files with your data.

## Highlights and use cases

By combining attachments with PowerTable's commenting and approval workflows, you can centralize feedback, track changes, collaborate on assets, and manage review and approval processes from a single location.

Attachment columns have the following characteristics:
* Each attached file can be up to **25 MB**.
* A record can contain multiple file attachments.
* You control access to attachments through the configured [row-level permissions](../powertable-how-to-set-up-access-control.md#row-access).
* Anyone with access to the record can view, download, and save attached files.

## Insert the attachment column

1. Go to **PowerTable** > **Insert Column** > **Visual Column** > **Add Attachment Column**.

    :::image type="content" source="../media/powertable-how-to-insert-columns/how-to-insert-attachment-columns/insert-attachment-column.png" alt-text="Screenshot of PowerTable Insert Column menu with Visual Column and Add Attachment Column highlighted." lightbox="../media/powertable-how-to-insert-columns/how-to-insert-attachment-columns/insert-attachment-column.png":::

1. Enter the column name and select **Save**. The process adds an attachment column.

    :::image type="content" source="../media/powertable-how-to-insert-columns/how-to-insert-attachment-columns/column-name.png" alt-text="Screenshot of PowerTable Add Attachment Column panel with Column Name field and Save button highlighted." lightbox="../media/powertable-how-to-insert-columns/how-to-insert-attachment-columns/column-name.png":::

## Attach a file

1. To attach a file to a record, select **+Add** from the **Attachments** column. Then select and drop the necessary file from your system.

    :::image type="content" source="../media/powertable-how-to-insert-columns/how-to-insert-attachment-columns/upload-file.png" alt-text="Screenshot of the Upload File window for attaching a file to a PowerTable record." lightbox="../media/powertable-how-to-insert-columns/how-to-insert-attachment-columns/upload-file.png":::

1. After uploading a file, select **Upload more** to upload more files to the record.

1. Select **Done** to finish adding your files.

    :::image type="content" source="../media/powertable-how-to-insert-columns/how-to-insert-attachment-columns/save-attachment.png" alt-text="Screenshot of the Upload File window showing a successfully uploaded PNG file with Upload more and Done buttons." lightbox="../media/powertable-how-to-insert-columns/how-to-insert-attachment-columns/save-attachment.png":::

The record now contains and shows the added attachments. Follow the same process to add files to other records. For a list of supported file types, see [Supported file types for attachments](../powertable-reference-supported-column-input-types.md#supported-file-types-for-attachment-columns).
