# Create conditional formatting rules in planning sheet

Conditional formatting helps you highlight important data points in a planning sheet by applying visual cues such as colors, icons, and styles based on defined rules. This improves readability and directs attention to key performance indicators in high-density tabular reports.

## Prerequisite

* You have access to the Planning sheet with the required dataset.

## Create a conditional formatting rule

1. In the Planning sheet navigate to **Format >** **Conditional formatting.**
1. Select **Create rule**.

    :::image type="content" source="media/planning-how-to-create-rules-conditional-formatting/create-rule.png" alt-text="Screenshot of creating rules in conditional formatting." lightbox="media/planning-how-to-create-rules-conditional-formatting/create-rule.png":::

1. Enter a **Title** for the rule.
1. Select where to apply formatting using **Apply to** (rows, columns, headers, or measures).

    :::image type="content" source="media/planning-how-to-create-rules-conditional-formatting/configure-rules.png" alt-text="Screenshot of configuring rules in conditional formatting." lightbox="media/planning-how-to-create-rules-conditional-formatting/configure-rules.png":::

1. Select the formatting scope using **Row hierarchy levels**:

* Values only
* Totals only
* Values and totals

    :::image type="content" source="media/planning-how-to-create-rules-conditional-formatting/row-hierarchy.png" alt-text="Screenshot of row hierarchy while configuring rules in conditional formatting." lightbox="media/planning-how-to-create-rules-conditional-formatting/row-hierarchy.png":::

1. Select a rule type from **Format by**:

* Rules (If conditions) - For more information about configuring rules using IF conditions, see [rules (IF conditions)](planning-how-to-format-rules-conditional-formatting.md)
* Color scale - For more information about configuring rules using Color scale, see [color scale](planning-how-to-format-color-scales-conditional-formatting.md)
* Classification - For more information about configuring rules Classification, see [Classification](planning-how-to-format-classification-conditional-formatting.md)

    :::image type="content" source="media/planning-how-to-create-rules-conditional-formatting/format-by.png" alt-text="Screenshot of format by while configuring rules in conditional formatting." lightbox="media/planning-how-to-create-rules-conditional-formatting/format-by.png":::

1. Select **Apply** to create the rule.
   
    :::image type="content" source="media/planning-how-to-create-rules-conditional-formatting/rule-created.png" alt-text="Screenshot of created rule in conditional formatting." lightbox="media/planning-how-to-create-rules-conditional-formatting/rule-created.png":::
