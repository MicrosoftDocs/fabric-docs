# Logical functions

Apply logical functions to test conditions and return *TRUE* or *FALSE* results. These functions are commonly used to evaluate expressions, validate data, and build conditional logic in reports.

## In this category

<table>
  <thead>
    <tr>
      <th width="138">Function</th>
      <th width="543">Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><a href="planning-logical-functions-and.md">AND</a></td>
      <td>Returns TRUE only if all conditions are TRUE.</td>
    </tr>
    <tr>
      <td><a href="planning-logical-functions-or.md">OR</a></td>
      <td>Returns <em>TRUE</em> if any condition is <em>TRUE</em>; returns <em>FALSE</em> only if all conditions are <em>FALSE</em>.</td>
    </tr>
    <tr>
      <td><a href="planning-logical-functions-not.md">NOT</a></td>
      <td>Returns the opposite logical value of the argument.</td>
    </tr>
    <tr>
      <td><a href="planning-logical-functions-xor.md">XOR</a></td>
      <td>Returns <em>TRUE</em> if the number of satisfied conditions is odd, and <em>FALSE</em> if it is even.</td>
    </tr>
    <tr>
      <td><a href="planning-logical-functions-in.md">IN</a></td>
      <td>Returns <em>TRUE</em> if the specified value exists in a list or array.</td>
    </tr>
    <tr>
      <td><a href="planning-logical-functions-isblank.md">ISBLANK</a></td>
      <td>Returns <em>TRUE</em> if the specified value is blank or empty.</td>
    </tr>
    <tr>
      <td><a href="planning-logical-functions-isblank.md">ISEMPTY</a></td>
      <td>Returns <em>TRUE</em> if the specified value is blank or empty. It works similar to ISBLANK.</td>
    </tr>
    <tr>
      <td><a href="planning-logical-functions-isnumber.md">ISNUMBER</a></td>
      <td>Returns <em>TRUE</em> if the specified value is a valid number.</td>
    </tr>
    <tr>
      <td><a href="planning-logical-functions-isrefexist.md">ISREFEXIST</a></td>
      <td>Returns <em>TRUE</em> if the specified reference exists in the model.</td>
    </tr>
  </tbody>
</table>
