# Creating a Planning Sheet

This article helps you create your Planning sheet.

## Prerequisites
* Create your connections to [connect to your database](plan/planning-create-database-connection.md)
* Create your connections to [connect to your semantic model](plan/planning-sheet-create-share-semantic-connection.md).
* A Fabric SQL database is automatically created in your workspace. This database stores your Planning sheet's metadata.

## Create a Planning sheet

1. Start in your Fabric workspace.
1. Create a **New item > Plan (preview)**.
   
  :::image type="content" source="media/planning-sheet-how-to-get-started/new-plan-1.png" alt-text="Screenshot of creating a new Plan (preview) item." lightbox="media/planning-sheet-how-to-get-started/new-plan-1.png":::

1. Name your plan and create it.

    :::image type="content" source="media/planning-sheet-how-to-get-started/new-plan-2.png" alt-text="Screenshot of providing name and location details for a new Plan.":::

1. Get your data from a semantic model or an Excel/CSV file, and then create a Planning sheet. Alternatively, you can create a Planning sheet and connect it to your data.

   :::image type="content" source="media/planning-sheet-how-to-get-started/new-plan-5.png" alt-text="Screenshot of the getting started options.":::

>[!NOTE]
> The database is automatically created when a Plan item is created.

## Connect a Planning sheet to a dataset

1. Select **Planning > Name the new Planning sheet** and **Create.

    :::image type="content" source="media/planning-sheet-how-to-get-started/new-plan-creation.png" alt-text="Screenshot of the getting started options.":::

1. Select **+Add** and **connect** to your semantic model connection.

    :::image type="content" source="media/planning-sheet-how-to-get-started/new-plan-creation-2.png" alt-text="Screenshot of adding semantic model connection.":::

1. Select the semantic model and select Add.

    :::image type="content" source="media/planning-sheet-how-to-get-started/new-plan-7.png" alt-text="Screenshot of choosing a semantic model.":::
      
1. Add semantic model data into your fields. Your first Planning sheet is now created.

   :::image type="content" source="media/planning-sheet-how-to-get-started/planning-sheet.png" alt-text="Screenshot of the created planning sheet." lightbox="media/planning-sheet-how-to-get-started/planning-sheet.png":::
