# Create a database connection for collaboration

This article explains how to connect to a database for collaboration from a Plan (preview) item. The connection is optional and allows viewers to collaborate on the plan.

## Prerequisites

You have access to Planning sheets.
You have permission to create and manage connections.

### Create a connection to a database for collaboration

1. Select the **Set up connection.**

     :::image type="content" source="media/planning-how-to-connect-to-database-collaboration/connection-setup.png" alt-text="Screenshot of database connection setup option for collaboration." lightbox="media/planning-how-to-connect-to-database-collaboration/connection-setup.png":::


1. Select **Create Connection** to create a Fabric SQL Connection or select an existing connection.

     :::image type="content" source="media/planning-how-to-connect-to-database-collaboration/create-connection.jpg" alt-text="Screenshot of creation of database connection  option for collaboration." lightbox="media/planning-how-to-connect-to-database-collaboration/create-connection.jpg":::

1. Select **Create new connection** from the Connection credentials dropdown.

     :::image type="content" source="media/planning-how-to-connect-to-database-collaboration/new-connection.png" alt-text="Screenshot of adding a new database connection  option for collaboration." lightbox="media/planning-how-to-connect-to-database-collaboration/new-connection.png":::


1. Enter a **Connection name**.
1. Select **Authentication kind** as **Organizational account.**

     :::image type="content" source="media/planning-how-to-connect-to-database-collaboration/connection-credentials.png" alt-text="Screenshot of adding a new database connection  option for collaboration." lightbox="media/planning-how-to-connect-to-database-collaboration/connection-credentials.png":::


1. Select **Create.**

A database connection is created to store collaboration data.
