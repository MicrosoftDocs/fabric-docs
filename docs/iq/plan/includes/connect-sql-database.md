## Create a connection to Fabric SQL database

1. When creating a new app, select a connection to connect to a Fabric SQL database. Select a connection and select **Connect**.

    :::image type="content" source="../media/includes/connect.png" alt-text="Screenshot of selecting the SQL connection.":::

1. Then, select a Fabric SQL database. The metadata for your app is stored here. Select **Add**.

    :::image type="content" source="../media/includes/select-database.png" alt-text="Screenshot of selecting the database.":::
