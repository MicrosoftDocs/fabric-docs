# Adding notes in Planning sheet

Notes are designed for **contextual explanation and documentation**, helping users interpret data, justify values, and improve decision-making. Planning enables **data-level annotation**, allowing users to document insights at granular levels such as rows, columns, or individual cells.

## Prerequisites

* You have access to a Planning sheet.

## Add a note to a cell

1. Select the cell to add a note in your Planning sheet.
1. Go to **Planning > Notes > Add New Note**.

    :::image type="content" source="media/planning-how-to-add-notes/add-note.png" alt-text="Screenshot of adding notes." lightbox="media/planning-how-to-add-notes/add-note.png":::

1. Enter your **note, apply formats,** and select **Save**.

     :::image type="content" source="media/planning-how-to-add-notes/format-notes.png" alt-text="Screenshot of formatting notes.":::

The note is now associated with the selected data point.

> [!NOTE]
> You can select multiple cells and add a single note that applies to all of them.
> :::image type="content" source="media/planning-how-to-add-notes/sheet-notes.png" alt-text="Screenshot of adding multiple notes." lightbox="media/planning-how-to-add-notes/sheet-notes.png":::

Hover over the Note indicator to see the note.

:::image type="content" source="media/planning-how-to-add-notes/hover-over-notes.png" alt-text="Screenshot of hovering over notes.":::

To edit an existing note, use the Pencil icon on the Note box. To delete an existing note, use the Delete icon on the Note box.

### Add notes at row or column level

1. Select the row or column header, and navigate to **Notes > Add New Note**.
1. **Enter a note** in the corresponding Notes field.

     :::image type="content" source="media/planning-how-to-add-notes/column-notes.png" alt-text="Screenshot of adding notes at column level." lightbox="media/planning-how-to-add-notes/column-notes.png":::

The note applies to the entire row or column and is visible across related data.

:::image type="content" source="media/planning-how-to-add-notes/row-notes.png" alt-text="Screenshot of adding notes at row level." lightbox="media/planning-how-to-add-notes/row-notes.png":::

## Add report-level notes

1. Go to **Planning > Notes > Report Summary**.

    :::image type="content" source="media/planning-how-to-add-notes/report-level-notes.png" alt-text="Screenshot of showing notes at report level.":::

1. **Enter your note** using the text editor.
1. **Apply formatting** such as:
   * Bullet points
   * Bold or italic text
   * Hyperlinks
1. **Save** your changes.

This note provides a high-level summary for the report.

:::image type="content" source="media/planning-how-to-add-notes/report-summary.png" alt-text="Screenshot of showing report summary." lightbox="media/planning-how-to-add-notes/report-summary.png":::

## Add footnotes

Footnotes display notes collectively at the bottom of the report for easy reference.

1. Add notes to cells, rows, or columns.
1. Open the **Notes** menu.
1. Enable the **Footnote** option.

    :::image type="content" source="media/planning-how-to-add-notes/footnotes.png" alt-text="Screenshot of adding footnotes.":::

All notes are displayed as numbered references in the footnote section.

:::image type="content" source="media/planning-how-to-add-notes/footnote-section.png" alt-text="Screenshot showing footnotes." lightbox="media/planning-how-to-add-notes/footnote-section.png":::

## Use the notes column

The notes column provides a structured way to enter notes for each row by providing a dedicated column to capture row-level annotations.

1. From the **Notes** menu, enable the **Notes column**.
2. Enter notes directly for each row.

    :::image type="content" source="media/planning-how-to-add-notes/notes-column.png" alt-text="Screenshot showing notes column." lightbox="media/planning-how-to-add-notes/notes-column.png":::

## Hide notes

You can hide notes to simplify the report view.

From the **Notes** menu, enable **Hide all notes**.

All notes are hidden from view, and note creation is disabled while this option is active.

:::image type="content" source="media/planning-how-to-add-notes/hide-notes.png" alt-text="Screenshot hiding notes." lightbox="media/planning-how-to-add-notes/hide-notes.png":::

## View all notes

A consolidated panel or view displays all notes associated with the report.

1. Go to the **Notes** menu.
2. Select **View all notes**.

   :::image type="content" source="media/planning-how-to-add-notes/view-notes.jpg" alt-text="Screenshot of viewing all notes." lightbox="media/planning-how-to-add-notes/view-notes.jpg":::

## Configure note settings

Use note settings to customize how notes appear and behave.

1. Select **Notes** > **Settings**.

    :::image type="content" source="media/planning-how-to-add-notes/note-settings.png" alt-text="Screenshot of note settings." lightbox="media/planning-how-to-add-notes/note-settings.png":::

2. Configure options such as:
   * Indicator display (style and size)
   * Indicator position within the cell
   * Rollup Indicator
   * Footnote Border
   * Footnote height
   * Marker Setting
3. Apply your changes.

You can adjust how note indicators are displayed and positioned within cells.

## Use marker mode

Marker mode allows you to highlight specific data points for emphasis. Marker mode lets you visually mark data for quick identification.

1. Go to **Notes > Marker Mode**.
2. Mark the cells, rows, or columns using the marker icon.
3. Use the **clear option** to remove markings if needed.
4. Customize color, shape, and width of markings from **Notes settings**.

    :::image type="content" source="media/planning-how-to-add-notes/marker-mode.png" alt-text="Screenshot of marker mode in notes." lightbox="media/planning-how-to-add-notes/marker-mode.png":::
