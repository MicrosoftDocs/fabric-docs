# Add notes in a planning sheet

You might find the notes feature of your planning sheets to be useful for contextual explanation and documentation. For example, with notes you can help interpret data, justify values, and improve decision-making. Planning enables data-level annotation, meaning that you can document insights at granular levels such as rows, columns, or individual cells.

[!INCLUDE [Fabric feature-preview-note](../../includes/feature-preview-note.md)]

## Prerequisite

Before you can add notes, ensure that you have access to a planning sheet.

## Add a note to a cell

1. Select the cell to add a note in your planning sheet.

1. Go to **Planning** > **Notes** > **Add New Note**.

    :::image type="content" source="media/planning-how-to-add-notes/add-note.png" alt-text="Screenshot of adding notes." lightbox="media/planning-how-to-add-notes/add-note.png":::

1. Enter your note, apply formats, and select **Save**.

     :::image type="content" source="media/planning-how-to-add-notes/format-notes.png" alt-text="Screenshot of formatting notes.":::

The note is now associated with the selected data point.

> [!TIP]
> You can select multiple cells and add a single note that applies to all of them.
> :::image type="content" source="media/planning-how-to-add-notes/sheet-notes.png" alt-text="Screenshot of adding a note to multiple cells." lightbox="media/planning-how-to-add-notes/sheet-notes.png":::

Hover over the note indicator to see the note.

:::image type="content" source="media/planning-how-to-add-notes/hover-over-notes.png" alt-text="Screenshot of hovering over a note indicator.":::

To edit an existing note, select the **Pencil** icon on the note box. To delete an existing note, select the **Delete** icon on the note box.

### Add notes at the row or column level

1. Select the row or column header, and then select **Notes** > **Add New Note**.

1. Enter a note in the corresponding notes field, and then select **Save**.

     :::image type="content" source="media/planning-how-to-add-notes/column-notes.png" alt-text="Screenshot of adding notes at the column level." lightbox="media/planning-how-to-add-notes/column-notes.png":::

The note applies to the entire row or column, and it's visible across related data.

:::image type="content" source="media/planning-how-to-add-notes/row-notes.png" alt-text="Screenshot of adding notes at the row level." lightbox="media/planning-how-to-add-notes/row-notes.png":::

## Add report-level notes

1. Go to **Planning** > **Notes** > **Report Summary**.

    :::image type="content" source="media/planning-how-to-add-notes/report-level-notes.png" alt-text="Screenshot of showing notes at report level.":::

1. Use the text editor to enter your note.

1. Apply formatting that you want, such as bullet points, bold or italic text, and hyperlinks.

1. Select **Save**.

This note provides a high-level summary for the report.

:::image type="content" source="media/planning-how-to-add-notes/report-summary.png" alt-text="Screenshot of showing a report summary." lightbox="media/planning-how-to-add-notes/report-summary.png":::

## Add footnotes

Footnotes display notes collectively at the bottom of the report for easy reference.

1. Add notes to cells, rows, or columns.

1. Open the **Notes** menu.

1. Turn on the **Footnote** option.

    :::image type="content" source="media/planning-how-to-add-notes/footnotes.png" alt-text="Screenshot of adding footnotes.":::

You see all notes displayed as numbered references in the footnote section.

:::image type="content" source="media/planning-how-to-add-notes/footnote-section.png" alt-text="Screenshot showing footnotes." lightbox="media/planning-how-to-add-notes/footnote-section.png":::

## Use the notes column

The notes column provides a structured way to enter notes for each row by providing a dedicated column to capture row-level annotations.

1. From the **Notes** menu, turn on the **Notes column**.

1. Enter notes directly for each row.

    :::image type="content" source="media/planning-how-to-add-notes/notes-column.png" alt-text="Screenshot showing a dedicated notes column." lightbox="media/planning-how-to-add-notes/notes-column.png":::

## Hide notes

You can hide notes to simplify the report view.

From the **Notes** menu, turn on **Hide all notes**.

All notes are hidden from view, and you can't create notes while this option is active.

:::image type="content" source="media/planning-how-to-add-notes/hide-notes.png" alt-text="Screenshot of hiding notes." lightbox="media/planning-how-to-add-notes/hide-notes.png":::

## View all notes

A consolidated panel or view displays all notes associated with the report.

1. Go to the **Notes** menu.

1. Select **View all notes**.

   :::image type="content" source="media/planning-how-to-add-notes/view-notes.jpg" alt-text="Screenshot of viewing all notes." lightbox="media/planning-how-to-add-notes/view-notes.jpg":::

## Configure note settings

Use note settings to customize how notes appear and behave.

1. Select **Notes** > **Settings**.

    :::image type="content" source="media/planning-how-to-add-notes/note-settings.png" alt-text="Screenshot of note settings." lightbox="media/planning-how-to-add-notes/note-settings.png":::

1. Configure options such as:
   * Indicator display (style and size)
   * Indicator position within the cell
   * Rollup indicator
   * Footnote border
   * Footnote height
   * Marker setting

1. Select **Apply**.

You can adjust how note indicators are displayed and positioned within cells.

## Use marker mode

You can use marker mode to highlight specific data points for emphasis. Marker mode lets you visually mark data for quick identification.

1. Go to **Notes** > **Marker Mode**.

1. Mark the cells, rows, or columns by using the marker icon.

1. Use **Clear Marker** to remove markings if needed.

1. In **Notes settings**, customize the color, shape, and width of markings as desired.

    :::image type="content" source="media/planning-how-to-add-notes/marker-mode.png" alt-text="Screenshot of marker mode in notes." lightbox="media/planning-how-to-add-notes/marker-mode.png":::
