# Configure display and formatting settings in PowerTable

Customize the appearance of your table by changing text formatting, spacing, row height, row highlighting, and gridline settings.

## Adjust row height

Increase or decrease the height of table rows to improve readability or display more data.

To adjust the row height:

1. Select the **Format** tab.
1. Use the **Row Height** control to increase or decrease the row height.

    :::image type="content" source="media/powertable-how-to-configure-display-formatting-settings/row-height.jpg" alt-text="Screenshot of PowerTable Format tab with the Row Height setting highlighted." lightbox="media/powertable-how-to-configure-display-formatting-settings/row-height.jpg":::

PowerTable updates the row height for the table.

## Customize text and header appearance

Use the **Display** tab to customize the appearance of table text, headers, and spacing.

To customize the display settings:

1. Select the **Format** tab.
1. Select **Appearance** on the toolbar. The **Appearance** pane opens and shows the **Display** tab by default.

    :::image type="content" source="media/powertable-how-to-configure-display-formatting-settings/display-settings.jpg" alt-text="Screenshot of the Appearance pane with the Display tab selected for customizing table text and headers." lightbox="media/powertable-how-to-configure-display-formatting-settings/display-settings.jpg":::

### Change text and header formatting

Customize the following text settings:

* **Font**: Select the font style for the table from the dropdown.
* **Font Size**: Adjust the size of the table text.
* **Text Color**: Change the color of cell values.
* **Header Orientation**: Show the header text in **Horizontal**, **Vertical**, **Clockwise**, or **Counterclockwise** orientations.
* **Header Text Color**: Change the color of the column headers.
* **Display Type in Header**: Show the configured input type with the column header.
* **Auto-Wrap Header Text**: Wrap long header names onto multiple lines instead of truncating them.

### Adjust table padding

Specify padding for the **Top**, **Bottom**, **Left**, and **Right** sides of the table. Padding values range from **0** to **100**.

## Customize gridlines and row highlighting

Switch to the **Gridlines** tab in the **Appearance** pane to customize the appearance of gridlines and row highlighting.

:::image type="content" source="media/powertable-how-to-configure-display-formatting-settings/gridlines-tab.png" alt-text="Screenshot of the Appearance pane Gridlines tab showing horizontal, vertical, header border, and row highlighting options." lightbox="media/powertable-how-to-configure-display-formatting-settings/gridlines-tab.png":::

### Configure gridlines

You can customize the following gridline settings:

* **Horizontal**: Display horizontal gridlines as **Solid**, **Dashed**, or **Off**. You can also customize the gridline color and specify the number of rows to skip between consecutive lines.
* **Skip Lines**: Specify the spacing between displayed horizontal gridlines.
* **Vertical**: Display vertical gridlines between columns as **Solid** or **Off**. When you turn on this setting, you can customize the gridline color.
* **Header Border**: Display borders for column headers and customize their color.
* **Show Outline**: Display a separator line below the column headers and customize its color.

### Highlight rows

Choose one of the following row highlight options:

* **Row Hover**: Highlights the row when you move the pointer over it. You can customize the hover text and background colors.
* **Row Alternate**: Highlights alternate rows in the table. You can customize the colors for odd and even rows.
* **Row Alternate Contrast**: Applies predefined contrasting colors to alternating rows, with customizable colors.
* **Row Alternate Flashy**: Applies predefined vibrant colors to alternating rows, with customizable colors.
* **Off**: Disables row highlighting.

:::image type="content" source="media/powertable-how-to-configure-display-formatting-settings/highlight-rows.png" alt-text="Screenshot of row highlight dropdown listing Row Hover, Row Alternate, Contrast, Flashy, and Off options.":::

## Reset to default styling

Select **Reset to Default Styling** to discard all appearance customizations and restore the default styling.

PowerTable restores all appearance settings, including text formatting, table padding, gridlines, row highlighting, and other display customizations, to their default values.
