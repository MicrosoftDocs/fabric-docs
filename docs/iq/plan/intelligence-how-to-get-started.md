# Get started with Intelligence sheets

The *Intelligence sheet* provides a no-code, end-to-end reporting experience that brings all your data visualization needs together on a single platform. This article details the steps to get started with your first Intelligence sheet.

[!INCLUDE [Fabric feature-preview-note](../../includes/feature-preview-note.md)]

## Create an Intelligence sheet

1. Select **New Intelligence Sheet** or select the Intelligence sheets icon from the landing page. Select the **Blank Canvas** option.

    :::image type="content" source="media/intelligence-how-to-get-started/new-intelligence-sheet.png" alt-text="Screenshot of creating a new intelligence sheet." lightbox="media/intelligence-how-to-get-started/new-intelligence-sheet.png":::

1. Select the icons on the right-hand side of the canvas to open the **Data** and **Visualizations** panes.

    :::image type="content" source="media/intelligence-how-to-get-started/data-visualizations-panes.png" alt-text="Screenshot of opening the data and visualization panes." lightbox="media/intelligence-how-to-get-started/data-visualizations-panes.png":::

1. Select a data presentation format (such as **Gantt**, **Matrix**, **KPI**, or **Chart**) from the **Visualizations** pane.

    :::image type="content" source="media/intelligence-how-to-get-started/visualizations-table.png" alt-text="Screenshot of selecting a format from the Visualizations tab.":::

1. Assign measures and dimensions to the visual to display it on the canvas.

    :::image type="content" source="media/intelligence-how-to-get-started/dimensions.png" alt-text="Screenshot of adding dimensions to create a visual." lightbox="media/intelligence-how-to-get-started/dimensions.png":::

## Import data from Excel and CSV files

1. Expand the **Queries** section in the **Data** pane.
1. Select **Add**, then choose the upload format (**Excel** or **CSV**).

    :::image type="content" source="media/intelligence-how-to-get-started/add-queries.png" alt-text="Screenshot of uploading a query showing Excel and CSV options.":::

1. Select the file for upload, then choose the sheet.

    :::image type="content" source="media/intelligence-how-to-get-started/upload-sheet.png" alt-text="Screenshot of selecting the file and sheet." lightbox="media/intelligence-how-to-get-started/upload-sheet.png":::

## Enable the toolbar

A key feature of the Intelligence sheet is its high level of customization. You can tailor reports to the specific needs of your audience. Each visual includes a dedicated toolbar that provides options for formatting, configuration, and accessing built-in features.

After adding the visual to the canvas and assigning data, hover over the visual and select the arrow icon to enable the toolbar.

## Add charts

1. Select the **Charts 100+** visual to add a chart.
1. Use the **Visualization** pane to search for a specific chart type, select an **IBCS** chart, or choose a chart from the **Special charts** section.
1. Select data from the **Data** and **Visualizations** panes to assign dimensions and measures. The chart is rendered in the canvas.

    :::image type="content" source="media/intelligence-how-to-get-started/render-chart.png" alt-text="Screenshot of a rendered chart showcasing dimensions from the Data and Visualization panes." lightbox="media/intelligence-how-to-get-started/render-chart.png":::

## Create tabular reports

1. Select the **Matrix** visual and assign row and column dimensions to create matrix tables. Matrix visuals support hierarchical dimensions in rows and columns.

    :::image type="content" source="media/intelligence-how-to-get-started/matrix.png" alt-text="Screenshot of a report with matrix visuals." lightbox="media/intelligence-how-to-get-started/matrix.png":::

1. Select the **Table** visual and assign row dimensions to create flat tables.

    :::image type="content" source="media/intelligence-how-to-get-started/tabular-report.png" alt-text="Screenshot of a report with flat tables." lightbox="media/intelligence-how-to-get-started/tabular-report.png":::
