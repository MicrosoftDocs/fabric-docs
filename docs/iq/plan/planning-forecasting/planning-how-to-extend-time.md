# Extend planning beyond source data periods

Data input fields are essential for planning, where targets, adjustments, and key drivers are entered to define budgets, forecasts, and scenarios. Use the data input to enter values for current and historical periods available in the source data.

The **Extend Time** feature combines data input and forecasting to enable data entry beyond the periods available in the source data.

## Prerequisites

* The planning sheet must have at least one data input measure.
* Forecasts shouldn't be created.

## Extend timelines for data input measures

1. Select **Extend Time**.
1. Select the start and end periods to extend the data input measures to the specified time range.
